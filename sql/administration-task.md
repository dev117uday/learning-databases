## Creating New User

![creating new user](../.gitbook/assets/create\_user.gif)

### **Options**

* **General**
  * Name : username
  * Comment : description
* **Definition**
  * Password : user password
  * Account Expires
  * Connection Limit : number of connections that user can have to DB, default -1
